# Sillyapp

A tiny glossy lane-crossing game mockup: Crossy Road energy, but much easier to win.

## Concept Map

- **Core loop:** move the character upward through a short set of lanes, reach the gold finish pad, get a ridiculous victory message, then tap again for another quick win.
- **Difficulty:** intentionally gentle. Obstacles are slow, collisions only nudge the player back, and the finish line is close.
- **Visual direction:** sleek arcade toy, glassy panels, lacquered buttons, neon reflections, soft shadows, animated shimmer, and a polished 3D-ish board.
- **Tone:** silly private-joke scoreboard rather than serious game narrative.
- **Win messages:**
  - `ella is extremely silly, hi T`
  - `ella is really, really silly`
  - `T is better than ella`
  - `hi T`
  - `ella ffs why are you here`

Note: the original first message was swapped for a clearly silly/non-defamatory version.

## Mockup

Open `index.html` in a browser. Use arrow keys, WASD, or the on-screen controls.
